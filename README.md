# خطة تسويقية - Adrenaline Fitness Wear

## نظرة عامة
مشروع ويب يعرض الخطة التسويقية المتكاملة لبراند Adrenaline Fitness Wear بشكل احترافي ومنسق.

## المميزات
- تصميم احترافي وجذاب
- ألوان متناسقة ومريحة للعين
- تنسيق كامل باللغة العربية (RTL)
- إمكانية الطباعة أو الحفظ كـ PDF
- Responsive Design (يعمل على جميع الأجهزة)

## كيفية الاستخدام
1. افتح ملف `index.html` في المتصفح
2. للحفظ كـ PDF: اضغط على زر "طباعة / حفظ كـ PDF" أو استخدم Ctrl+P

## الملفات
- `index.html` - الصفحة الرئيسية
- `style.css` - ملف التنسيق
- `Adrenaline_Marketing_Plan.md` - الملف الأصلي

## التقنيات المستخدمة
- HTML5
- CSS3
- Responsive Design
- Print-friendly Layout
